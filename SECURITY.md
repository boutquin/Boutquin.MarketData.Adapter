# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in Boutquin.MarketData.Adapter, report it responsibly.

Do not open a public issue for security vulnerabilities.

Please use GitHub Security Advisories to report vulnerabilities privately:
https://github.com/boutquin/Boutquin.MarketData.Adapter/security/advisories/new

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline

- Acknowledgment: within 48 hours
- Initial assessment: within 7 days
- Fix or mitigation: target within 30 days, depending on severity

### Scope

This policy covers Boutquin.MarketData.Adapter NuGet packages and source code. Third-party dependencies (for example .NET runtime and NuGet dependencies) should be reported to their maintainers.
